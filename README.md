# Keegan Labrot | Frontend Developer Portfolio

A modern responsive frontend portfolio website built with HTML, CSS, and JavaScript.
This project showcases clean UI design, responsive layout techniques, interactive navigation, and a custom Apple-inspired liquid glass visual style.

## Overview

This portfolio was created to present my frontend development skills, project experience, and contact information in a polished, professional format.

The design focuses on:

* Responsive layouts for desktop, tablet, and mobile
* Clean visual hierarchy
* Modern glass-style UI components
* Mobile hamburger navigation
* Smooth scrolling
* Interactive buttons and cards
* Accessible semantic HTML structure

## Features

* Responsive sticky navigation bar
* Mobile hamburger menu
* Apple-inspired liquid glass UI styling
* Animated ambient background elements
* Hero section with developer introduction
* About section
* Skills grid
* Projects section
* Contact form layout
* Footer with navigation and social/contact links

## Sections

### Home

A hero section introducing me as a frontend developer and summarizing the type of websites I build.

### About

A short personal overview explaining my focus on polished, accessible, and responsive web experiences.

### Skills

A visual grid of frontend development skills, including HTML, CSS, JavaScript, responsive design, GitHub, UI design, DOM manipulation, and accessibility.

### Projects

A collection of frontend project cards showing examples of my work and design direction.

### Contact

A simple contact form layout with links for email, GitHub, and LinkedIn.

## Design Style

The interface uses a clear liquid-glass aesthetic inspired by modern Apple-style UI design.
The glass effect is created with layered gradients, transparent backgrounds, backdrop blur, thin rim highlights, inner shadows, and soft ambient background lighting.

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/RegionZeroProcedural/PortfolioPage.git
```

2. Open the project folder:

```bash
cd your-repository-name
```

3. Open `index.html` in your browser.

You can also use the Live Server extension in Visual Studio Code for local development.
